## Python

```
    Usage:      
        python3 downloader.py -n 斗破苍穹 -p proxy

        -h show help
        -n --name <Book Name>
        -p --proxy <Proxy File Path>
```

## Golang
```
> go build
> ./main -h

Usage of ./main:
  -name string
    	请输入书名，宁可少字不能错字
  -proxy string
    	请输入代理文件路径，强烈建议使用代理
```
